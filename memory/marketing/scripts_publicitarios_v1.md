## ✅ Lo que vos hacés con esto

1. **CapCut Pro / Premiere Pro / DaVinci**: importás los textos como subtítulos.
2. **ElevenLabs** o voz IA de CapCut: pegás los voiceovers, exportás los MP3.
3. **Banco de stock**: bajás b-rolls de Pexels / Mixkit / Storyblocks (gratis y de pago).
4. **Captura de pantalla**: 5 capturas de la propia Lluvia App Studio funcionando (landing, chat con agente, push exitoso, cobro PayPal, dashboard de balance).
5. **Editor**: cortás, pegás los textos como overlay, sincronizás voiceover, exportás vertical 9:16 para TikTok/Reels y horizontal 16:9 para YouTube.

**Producción total estimada**: 8-12 horas para los 8 reels + 3 videos largos si lo hacés solo, 2-4 horas con un editor profesional ($15-40 USD/hora).
